# 安全策略 / Security Policy

## 报告安全问题 / Reporting a Vulnerability

BetterBackup 是一个数据安全敏感的备份 mod。如果你发现了可能导致备份损坏、数据丢失、
静默数据不一致或其它安全相关的问题，请**私下**报告，不要先公开开 issue，以便在修复发布
前不暴露给他人。

BetterBackup is a data-safety-sensitive backup mod. If you find a vulnerability that could
lead to backup corruption, data loss, silent data inconsistency, or any other security-relevant
issue, please report it **privately** rather than opening a public issue first, so it is not
disclosed before a fix is available.

报告渠道 / How to report:

- 首选：通过本仓库的 GitHub Security Advisories 私下报告
  (Preferred: GitHub Security Advisories on this repository) —
  https://github.com/xiaoxiao-cvs/Shinoyuki-BetterBackup/security/advisories/new

请在报告中尽量包含 / Please include when possible:

- 受影响的版本 / affected version(s)
- 复现步骤或最小复现 / reproduction steps or a minimal repro
- 影响范围（数据丢失 / 损坏 / 崩溃等）/ impact (data loss / corruption / crash, ...)
- 相关日志、堆栈或快照/store 状态 / relevant logs, stack traces, or snapshot/store state

## 支持范围 / Scope

- 请只报告当前主分支 (main) 或最新构建中的问题。
  Please report issues against the current main branch or the latest build.
- 常规 bug、功能请求、使用问题请走公开 issue 模板，不属于本安全渠道。
  Ordinary bugs, feature requests, and usage questions belong in the public issue templates,
  not this security channel.
